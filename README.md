# Claude Marketplace

Personal Claude Code plugin marketplace.

## Usage

Add this marketplace to Claude Code:

```
/plugin marketplace add ethanmcrae/claude-marketplace
```

Then install plugins:

```
/plugin install linear-workflow@ethanmcrae-marketplace
```
