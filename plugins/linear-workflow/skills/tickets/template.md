# Tickets Dashboard Template

Format the output exactly like this. Omit a section entirely if it has no tickets.

---

## In Progress

| Ticket | Title | Est | Project |
|--------|-------|-----|---------|
| ETH-XX | ...   | M   | ...     |

## Todo

| Ticket | Title | Est | Project |
|--------|-------|-----|---------|
| ETH-XX | ...   | S   | ...     |

---

Rules:
- Use the estimate labels (XS=1, S=2, M=3, L=5, XL=8), not raw numbers
- Sort each section by priority (urgent first), then by estimate (largest first)
- If both sections are empty, say "No active tickets."
