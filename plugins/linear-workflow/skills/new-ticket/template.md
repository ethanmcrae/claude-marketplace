# New Ticket Reference

## Estimate Scale (use these labels when asking the user)

| Label | Points | Guidance                        |
|-------|--------|---------------------------------|
| XS    | 1      | Trivial — typo, config tweak    |
| S     | 2      | Small — single-file change      |
| M     | 3      | Medium — a few files, < 1 day   |
| L     | 5      | Large — multi-file, ~1-2 days   |
| XL    | 8      | Extra large — multi-day effort  |
